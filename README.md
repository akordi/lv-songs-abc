
## Prerequisites

- [Go](https://golang.org/dl/)
- [Hugo](https://gohugo.io/getting-started/installing/)
- [Git](https://git-scm.com/downloads)


Downloading content

`go run download.go` 

Downloading theme

`git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1` 

Running site

`hugo serve`
